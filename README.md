# Cover Depan-Belakang Maker

Privacy-first browser utility for merging a front cover PDF, a main PDF, and a back cover PDF into one downloaded PDF.

## Core Scope

- Fully browser-only PDF processing.
- No user PDF upload to a server.
- Default cover stock for Presentasi, Dokumen, and Canva.
- Local uploaded cover stock stored on the user's device with IndexedDB.
- Output filename format: `OriginalName_Proses.pdf`.

## Commands

```bash
npm run dev
npm run test
npm run typecheck
npm run lint
npm run build
```
